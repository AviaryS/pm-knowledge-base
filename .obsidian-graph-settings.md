# Настройки графа для Obsidian

## Рекомендуемые настройки графа

### Цвета узлов по тегам
- `#agile` - Синий
- `#scrum` - Зеленый
- `#matrix` - Оранжевый
- `#template` - Фиолетовый
- `#practice` - Красный
- `#learning-plan` - Желтый
- `#methodology` - Голубой

### Размер узлов
- Центральные узлы (README, INDEX) - Большие
- Методологии - Средние
- Матрицы и инструменты - Средние
- Шаблоны - Малые
- Практики - Малые

### Фильтры для графа

#### Показать только методологии
```
tag:#methodology OR tag:#agile OR tag:#scrum
```

#### Показать только матрицы
```
tag:#matrix OR tag:#tool
```

#### Показать связи между разделами
```
path:"01 Методологии" OR path:"02 Матрицы" OR path:"03 Планы" OR path:"04 Шаблоны" OR path:"05 Практики"
```

## Структура графа

### Центральные узлы
- `README.md` - главная точка входа
- `INDEX.md` - индекс навигации

### Основные разделы
- `01 Методологии/` - методологии управления
- `02 Матрицы и инструменты/` - практические инструменты
- `03 Планы обучения/` - программы обучения
- `04 Шаблоны документов/` - готовые шаблоны
- `05 Практики и техники/` - практические навыки

## Советы по навигации

1. Используйте теги для фильтрации в графе
2. Следуйте ссылкам между документами
3. Используйте INDEX.md для быстрого поиска
4. Настройте цвета узлов по тегам для лучшей визуализации
