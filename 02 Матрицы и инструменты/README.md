# Матрицы и инструменты PM

## Матрицы для анализа и принятия решений

### [[Stakeholder Matrix|Матрица стейкхолдеров]]
Анализ заинтересованных сторон по влиянию и интересу

### [[RACI Matrix|RACI матрица]]
Определение ролей и ответственности в проекте

### [[Priority Matrix|Матрица приоритетов]]
Определение приоритетов задач (MoSCoW, Value/Effort)

### [[Risk Matrix|Матрица рисков]]
Оценка и приоритизация рисков проекта

### [[Impact Effort Matrix|Матрица Impact/Effort]]
Оценка задач по влиянию и усилиям

## Другие инструменты

- **User Story Mapping** - карта пользовательских историй
- **Burndown Chart** - график выполнения работы
- **Velocity Chart** - график скорости команды
- **Cumulative Flow Diagram** - диаграмма накопленного потока
- **Gantt Chart** - диаграмма Ганта

## Когда использовать

- **Stakeholder Matrix** - в начале проекта для понимания участников
- **RACI** - при распределении ответственности
- **Priority Matrix** - при планировании спринта/релиза
- **Risk Matrix** - при управлении рисками
- **Impact/Effort** - при выборе задач для бэклога
