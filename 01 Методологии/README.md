# Методологии управления проектами

## Основные методологии

### [[Agile|Agile]] 
Гибкая методология разработки, основанная на итерациях и обратной связи

### [[Scrum|Scrum]]
Фреймворк для реализации Agile с ролями, артефактами и событиями

### [[Kanban|Kanban]]
Метод визуализации работы и управления потоком задач

### [[Scrumban|Scrumban]]
Гибрид Scrum и Kanban

### [[Waterfall|Waterfall]]
Каскадная модель разработки с последовательными этапами

### [[SAFe|SAFe]]
Scaled Agile Framework для больших организаций

## Сравнение методологий

| Методология | Когда использовать | Гибкость | Документация |
|------------|-------------------|---------|--------------|
| Waterfall | Четкие требования | Низкая | Высокая |
| Agile | Изменяющиеся требования | Высокая | Средняя |
| Scrum | Команды 3-9 человек | Высокая | Средняя |
| Kanban | Непрерывный поток | Очень высокая | Низкая |
